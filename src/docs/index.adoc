== Диаграмма

Вы можете вставлять диаграммы Mermaid прямо в текст.

[mermaid]
----
graph TD;
    A[Начать] --> B{Есть изменения?};
    B -- Да --> C[Закоммитить];
    C --> D[Запушить];
    B -- Нет --> E[Отдыхать];
    D --> E;
----
